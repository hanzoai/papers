\section{Related Work}


API gateway design has evolved significantly. Kong \cite{kong2019} provides plugin-based extensibility. AWS API Gateway \cite{aws2019} offers serverless integration. GraphQL \cite{graphql2015} introduced flexible querying, with Apollo Server \cite{apollo2019} providing production implementations.

Rate limiting algorithms are well-studied. The token bucket \cite{turner1986} and leaky bucket \cite{bellovin1999} provide complementary approaches. Our distributed implementation draws from Redis-based patterns \cite{redis2019}.
