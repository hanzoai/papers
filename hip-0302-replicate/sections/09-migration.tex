\section{Migration}


\subsection{Existing PVC-Backed Services}

Services currently using PVCs migrate to replicate in three steps:

\begin{enumerate}[nosep]
    \item \textbf{Bootstrap.} Deploy the replicate sidecar alongside the
    existing PVC-backed pod. The sidecar begins replicating the PVC-resident
    database to S3. Wait for the initial snapshot to complete.

    \item \textbf{Cutover.} Redeploy the StatefulSet with \texttt{emptyDir}
    instead of PVC. The init container restores from S3. Verify the restored
    database matches the PVC-resident database (row count, checksum).

    \item \textbf{Cleanup.} Delete the PVCs. Update the service's Kustomize
    overlay to remove PVC-related resources.
\end{enumerate}

\subsection{New Services}

New Base services created after this HIP include the replicate sidecar in
their initial Kustomize base. No migration required.
