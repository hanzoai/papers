\section{Integration with Hanzo Commerce}

\label{sec:integration}
% ============================================================================

The crypto payment extension is a module within the Hanzo Commerce stack:

\begin{itemize}[leftmargin=1.4em]
    \item \textbf{Hanzo IAM}: Merchant authentication and API key
    management. OAuth 2.0 with org-scoped permissions.
    \item \textbf{Hanzo KMS}: Encryption of MPC key material at rest.
    All merchant key shares encrypted with KMS-managed keys.
    \item \textbf{Hanzo Base}: Merchant configuration, payment records,
    and settlement history stored in Base (SQLite for local dev,
    PostgreSQL for production multi-instance).
    \item \textbf{Hanzo Gateway}: Rate limiting, request routing, and
    TLS termination for all merchant API traffic.
    \item \textbf{Hanzo Webhooks}: Event delivery for payment lifecycle
    events with guaranteed at-least-once delivery and HMAC signature
    verification.
\end{itemize}
