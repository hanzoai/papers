\section{Instant Settlement on Lux}

\label{sec:lux}
% ============================================================================

For payments on Lux C-Chain, settlement is near-instant:

\begin{itemize}[leftmargin=1.4em]
    \item \textbf{Finality}: Lux consensus provides sub-second deterministic
    finality. Once a transaction is confirmed, it cannot be reverted.
    \item \textbf{Gas cost}: $<$\$0.01 per transaction.
    \item \textbf{USDC native}: USDC is deployed natively on Lux C-Chain,
    avoiding bridge risk.
    \item \textbf{Merchant notification}: Webhook fired to merchant
    within 2 seconds of payment.
\end{itemize}

For merchants prioritizing settlement speed, the checkout SDK
preferentially suggests Lux C-Chain as the payment network while
maintaining support for all configured chains.
