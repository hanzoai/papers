\section{HANZO Token}

\label{sec:token}
% ============================================================================

\begin{definition}[HANZO Token]
HANZO is the native utility token of Hanzo Network with a fixed maximum supply of
$S_{\max} = 1{,}000{,}000{,}000$ (one billion) tokens, 18 decimal places, and no
minting capability after genesis.
\end{definition}

\subsection{Token Functions}

HANZO serves four distinct economic functions:

\begin{enumerate}[leftmargin=1.4em]
    \item \textbf{Compute payment}: All AI inference, training, and fine-tuning
    jobs on Hanzo Network are priced and settled in HANZO.
    \item \textbf{Node staking}: Inference node operators must stake HANZO as
    collateral, subject to slashing for incorrect results.
    \item \textbf{Governance}: HANZO holders vote on model selection,
    infrastructure upgrades, parameter changes, and treasury allocation.
    \item \textbf{Fee reduction}: Holding HANZO in a staking contract reduces
    API fees on platform.hanzo.ai by up to 50\%.
\end{enumerate}
