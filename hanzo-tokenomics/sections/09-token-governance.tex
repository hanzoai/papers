\section{Token Governance}

\label{sec:tokengov}
% ============================================================================

The Hanzo DAO governs all token-economic parameters. Governance is implemented as
an on-chain contract on Hanzo Network (chain ID 36963) with the following
structure:

\begin{itemize}[leftmargin=1.1em]
    \item \textbf{Proposal threshold}: 100,000 HANZO (0.01\% of supply) to
    submit a proposal.
    \item \textbf{Quorum}: 50,000,000 HANZO (5\% of supply) must participate for
    a vote to be valid.
    \item \textbf{Approval}: $>$66\% of voting power must approve.
    \item \textbf{Timelock}: 7-day delay between approval and execution.
    \item \textbf{Security council}: 5-of-9 multi-sig with veto power and
    emergency pause capability.
\end{itemize}
