% ABSTRACT
% ============================================================================
\begin{abstract}
We present Hanzo Crypto Payment Rails, an extension of the Hanzo Commerce
platform that enables merchants to accept cryptocurrency payments (BTC,
ETH, USDC, USDT) with instant settlement on Lux Network. Built on the
foundation of Hanzo's checkout.js library (in production since 2015),
the crypto payment extension uses MPC threshold signing for merchant
wallet management, multi-chain transaction monitoring (EVM, Bitcoin,
Solana), and automatic fiat off-ramps. We describe the architecture:
payment channel creation, multi-chain address derivation via a single MPC
key ceremony, real-time exchange rate locking, settlement finality
guarantees, and integration with the Hanzo Commerce API. Production
benchmarks from merchant deployments show median payment confirmation in
1.2 seconds on Lux, 12 minutes on Bitcoin, and 0.4 seconds on Solana,
with settlement to merchant fiat accounts within 24 hours.
\end{abstract}
